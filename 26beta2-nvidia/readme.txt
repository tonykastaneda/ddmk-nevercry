Instructions:
- Extract the contents of the zip file inside the DMC HD Collection directory
(default location: C:\Program Files (x86)\Steam\steamapps\common\Devil May Cry HD Collection) or go to Steam Library > Right Click DMC HDC > Manage > Browse Local Files.

If the DMC3 game doesn't open, install the following:
- Microsoft Visual C++ Redistributable x64 and x86
https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170

- DirectX End-User Runtimes (June 2010)
https://www.microsoft.com/en-us/download/details.aspx?id=8109
